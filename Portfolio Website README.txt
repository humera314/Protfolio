Portfolio Website README

To view this portfolio website, follow these steps:

1. Download or clone the repository to your local machine.

2. Open the downloaded folder in a text editor or IDE of your choice.

3. Make sure you have a web browser installed on your machine (e.g., Google Chrome, Mozilla Firefox, Safari).

4. Locate the "index.html" file in the downloaded folder and double-click to open it in your web browser. This will open the portfolio website locally on your machine.

5. When the page open it will show just name so explore the different sections of the website by clicking on the navigation links in the header or scrolling through the main content sections.


6. Enjoy exploring the portfolio website!

Note: This portfolio website is designed to be responsive and adjust its layout for different screen sizes. You can test its responsiveness by resizing your browser window or viewing it on different devices.
